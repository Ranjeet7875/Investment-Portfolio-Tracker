Level Question Key Points
Basic What is React? - JavaScript library for UIs
- Component-based
- Uses Virtual DOM
Components in React - Building blocks of React apps
- Can be functional or class
JSX - Syntax extension for JavaScript
- Looks like HTML
- Gets compiled to React.createElement()
Intermediate Controlled vs Uncontrolled - Controlled: React manages state
- Uncontrolled: DOM manages state
React Keys - Help identify list items
- Should be unique
- Avoid using index
Performance Optimization - React.memo
- useMemo/useCallback
- Code splitting
Advanced React Fiber - Reconciliation engine rewrite
- Enables incremental rendering
Hooks Rules - Only call at top level
- Only call from React functions
useContext vs Redux - useContext for simple state
- Redux for complex apps
